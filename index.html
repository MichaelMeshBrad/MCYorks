Get started in Yorkshire

Get a device

Pick a LoRa-capable device supported by MeshCore.

Flash MeshCore

Install the latest stable firmware. Use the official web flasher.

Set EU/UK "Narrow" Preset

Use Freq:869.618, BW:62.5, SF:8, CR:8 for compatibility across Yorkshire.

Place your node

Higher is better. Windowsills, lofts, hilltops—line-of-sight helps your neighbours.

Join the “Yorkshire” channel

Within the app, go to the Add Channel section, choose Join Hashtag Channel, and in the channel name box, enter yorkshire (it’s a # channel).

Say hello

Send a message to default "Public" Channel and say Hi!, don't forget to spread the word!
