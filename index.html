Meshcore Yorkshire

Community-run MeshCore group across Yorkshire

View live map How to get started Facebook Group
Get connected• Help expand coverage• Stay resilient

MeshCore Yorkshire is a volunteer effort to build practical, low‑power radio coverage across the county for short, secure text messaging and situational awareness — useful at events, on the hills, and when phone networks are busy or down.

Get started in Yorkshire

Get a device

Pick a LoRa-capable device supported by MeshCore.

Flash MeshCore

Install the latest stable firmware. Use the official web flasher.

Set EU/UK defaults

Use the 868 MHz default public channel for compatibility across Yorkshire.

Place your node

Higher is better. Windowsills, lofts, hilltops—line-of-sight helps your neighbours.

Say hello

Send a message, show up on the map, and help others hop across the county.
