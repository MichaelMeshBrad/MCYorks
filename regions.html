📡 Overview

Regions control how messages are forwarded across the MeshCore network. Setting a region ensures traffic stays local and prevents unnecessary flooding.

🔧 Repeater Setup

Requirements

  • Firmware version 1.15.0 or newer

Configuration Steps

1. Log in to the repeater management interface
2. Open Regions
3. Click the ⋮ (top right)
4. Select Default Region
5. Enter yorkshire
6. Click the ✓ tick icon to save

Restrict Unscoped Traffic

1. Find Packets without region scope
2. Click ⋮ (top right)
3. Select Deny Flood

📱 Companion Setup

Per Channel

  • Open a channel
  • Use the top-right menu
  • Set the region/scope

Per Device (Experimental)

  • Go to Settings → Experimental
  • Assign a default region

💡 Notes

  • Region must be lowercase
  • All repeaters should match region naming
  • Always confirm changes with the ✓ save icon

Correct setup prevents unnecessary rebroadcasting and keeps the network efficient.
